# AI-Based Smart Visitor Gate Pass System Using Raspberry Pi

## Overview

An AI-powered visitor management system built using Raspberry Pi.

## Features

- Visitor Registration
- Face Recognition
- QR Code Generation
- Digital Gate Pass
- SQLite Database Storage
- Raspberry Pi Camera Support

## Technologies Used

- Python
- Raspberry Pi
- OpenCV
- Face Recognition
- SQLite
- QR Code

## Installation

```bash
git clone https://github.com/yourusername/AI-Smart-Visitor-Gate-Pass.git

cd AI-Smart-Visitor-Gate-Pass

pip install -r requirements.txt

python main.py
```

## Hardware Required

- Raspberry Pi 4
- Pi Camera Module
- Monitor
- Keyboard
- SD Card
